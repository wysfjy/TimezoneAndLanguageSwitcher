# 时区语言一键切换 (Timezone & Language Switcher)

一款专为 Android 打造的高级系统工具，支持通过 **Shizuku** 或 **Root** 权限快速修改系统时区及单个应用的语言设置。特别针对多用户（如小米 XSpace/应用分身）进行了深度优化。

## 🌟 主要功能

- **🌍 应用语言切换**：独立设置每个应用的显示语言，支持一键重置为系统默认。
- **🕙 全球时区设置**：
    - **基础模式**：精选常用时区（中国、香港、伦敦等）。
    - **高级模式**：解锁全球所有时区搜索。
- **👥 多用户/分身支持**：完美支持“应用分身”（XSpace）等独立空间的应用管理与语言设置。
- **🚀 智能激活**：
    - 启动时自动检测 Root 权限并尝试静默激活。
    - 支持 Shizuku 权限授权，一键进入工作状态。
    - 记忆上次激活模式，真正做到“开箱即用”。
- **🎯 自动化操作**：修改应用语言后自动强行停止（Force Stop）目标应用，确保设置立即生效。

## 🛠 权限要求

本应用属于系统级工具，需要以下任一权限即可运行：
1. **Shizuku (推荐)**：通过 ADB 或无线调试运行。
2. **Root 权限**：传统的超级用户访问权限。

## 📥 安装与使用

1. 下载并安装最新的 APK。
2. 确保已开启 **Shizuku** 或已授予 **Root** 权限。
3. 首次进入请点击“通过 Shizuku 激活”或“通过 Root 激活”。
4. 在应用列表中找到目标应用，选择目标语言即可。

## 📅 发行日志

### v1.0.0 (Initial Release - 2026.06.11)

这是本工具的首次正式发布，带来了完整的核心功能体验。

#### 新增特性
- **[核心]** 实现了基于 `cmd locale` 命令的非侵入式应用语言修改方案。
- **[多用户]** 增加分身空间（UserID: 999 等）识别逻辑，解决分身应用无法修改语言的痛点。
- **[UI/UX]** 
    - 采用了 Material 3 现代设计语言，支持深色模式。
    - 增加了应用加载、权限检测的进度条反馈。
    - 优化了语言修改后的反馈逻辑：针对非系统应用自动执行强停。
- **[图标]** 引入了全新的蓝青色渐变图标，涵盖地球、时钟与切换元素。
- **[稳定性]** 
    - 针对 Android 13+ 的 `QUERY_ALL_PACKAGES` 权限进行了适配。
    - 修复了 Shizuku 在某些系统下无法授权导致闪退的问题。
    - 增加了启动自检逻辑，实现 Root 环境下的免操作进入。

#### 修复与优化
- 优化了时区搜索的匹配速度。
- 修复了将语言设置回“默认”时可能出现的参数解析错误。
- 增加了针对系统应用的特别提醒弹窗。
